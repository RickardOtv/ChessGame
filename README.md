# ChessGame

A web-based chess game. Very Cool.

## Description

This is a chess game that runs in your web browser, allowing you to play chess online.

## Getting Started

### Installation

1. Clone the repository
```bash
git clone https://github.com/RickardOtv/ChessGame.git
```

2. Navigate to the project directory
```bash
cd ChessGame
```

3. Open the game in your browser
```bash
open index.html
```

## How to Play

Click on a piece to select it, then click on a valid square to move it. Follow standard chess rules to win!
